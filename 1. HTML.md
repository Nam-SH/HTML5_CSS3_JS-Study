# HTML 기본 요소



## 1. 구획 루트

```html
<h1>, <h2>, <h3>, <h4>, <h5>, <h6>: 6단계의 문서 제목을 구현한다.
```



## 2. 문자 콘텐츠

```html
<div>: 본질적으로 아무것도 나타내지 않는 플로우 컨텐츠의 일반적인 컨테이너이다
<figcaption>: <figure> 요소가 포함하는 도표나 도해 등의 다른 데이터에 대해 그에 관련된 설명 또는 범례를 표현
<figure>: 독립적인 콘텐츠를 표현한다.
<hr>: 문단 레벨 요소들 사이에서 주제의 분리를 나타낸다.
<ul>: 리스트에서의 순서가 의미없는, 숫자 순서를 가지고 있지 않은, 정렬되지 않은 항목들의 리스트를 나타낸다.
<ol>: 정렬된 리스트의 항목들을 나타낸다.
<li>: 리스트 항목을 나타낼때 사용된다.
<p>: 텍스트의 단락을 나타낸다.
<pre>:  미리 서식이 지정된 텍스트를 나타낸다.
```



## 3. 인라인 텍스트 시멘틱

```html
<a>: 다른 페이지, 파일, 같은 페이지의 어느 위치,그 외 다른 URL로 연결할 수 있는 하이퍼링크를 만든다.
<br>: 텍스트에서 줄바꿈을 생성한다.
<em>: 강조를 표시할때 사용한다.
<strong>: 글자에 강조 효과를 줄 수 있으며, bold 체로 보여지게 
<i>: 특정 이유로 인하여 평범한 글자와 구분하기 위해 사용된다.
<span>: 컨텐츠를 파싱하기 위한 제네릭 인라인 컨테이너이다.
```



## 4. 이미지 & 멀티미디어

```html
<img>: 문서에 이미지를 넣는다.
<audio>: 문서에 소리 컨텐츠를 포함하기 위해 사용한다.
    aotoplay - 웹 브라우저에서 음원을 자동으로 재생
    loop - 웹 브라우저에서 음원 반복 재생
    src - 재생할 음원 존재하는 URL 입력
    controls - 웹 브라우저에 음원재생을 제어하는 제어기 생성
    preload - 사용자가 사용할 생각이 없더라도 음원을 미리 다운
    volume - 음원의 재생 볼륨을 설정 (0.0 ~ 1.0까지) 
<video>: 동영상 콘텐츠를 포함하기 위해서 사용된다.
    autoplay  - 자동으로 비디오 재생
    src - 재생할 비디오 경로 지정
    width, height - 비디오 재생 너비 또는 높이 지정
    controls - 비디오 재생을 제어하는 표시 생성
    loop - 비디오 반복 재생
    poster - 비디오 다운로드 표시
    preload - 사용자 생각이 아니더라도 오디오를 다운로드
    muted - 비디오 오디오 출력을 중지 
```



## 5. 내장 콘텐츠

```html
<iframe>: 다른 HTML 페이지를 현재 페이지에 포함시키는 중첩된 브라우저 컨텍스트를 나타낸다.
	src - iframe에 삽입될 문서의 주소
	width - iframe의 너비 지정 (px, % 가능)
	height - 높이 지정(px, % 가능)
	frameborder - 테두리를 표시할지 말지를 지정 - 1(테두리가 있음) 0(테두리 지우기)
	scrolling - 스크롤바 유무를 선택 (yes(스크롤바 표시), no(스크롤바 없앰), auto (자동))
	longdesc - iframe에 담길 내용을 설명하는  페이지.
	marginheight - 내용의 top(위) bottom(아래) margin.
	marginwidth - 내용의 left(좌) right(우) margin.
	align - iframe을  정렬 (top( 맨 위) middle(중앙) bottom(아래) left(왼쪽) right(오른쪽))
	name - taget이 필요한 프레임의 이름 (id와 같음)
```

## 6. 수정

```html
<del>: 문서에서 삭제된 텍스트의 범위를 나타낸다. 이 요소는 보통 가운데줄 텍스트로 렌더링 된다.
<ins>: 문서에서 추가된 글자의 범위를 나타낸다.
```



## 6. 표 콘텐츠

```html
<caption>: 표의 제목을 나타낸다.
<col>: 표 내부의 열을 정의하며 모든 공통 셀들의 공통 시멘틱을 정의하기 위해 사용된다.
<colgroup>: 표 내부의 칸의 그룹을 정의한다.

<table>: 테이블을 만드는 태그
    표의 부모 요소에 overflow-x: auto; 속성을 추가하면 표에서만 가로 스크롤이 생긴다.(반응형)
<th>: 테이블의 헤더부분을 만드는 태그이다.
<tr>: 테이블의 행(row, 가로)을 만드는 태그이다.
<td>: 테이블의 열(col, 세로)을 만드는 태그이다.
    colspan: 가로로 이웃한 셀을 합칠 때 사용한다.
    rowspan: 세로로 이웃한 셀을 합칠 때 사용한다.
```



## 7. 폼

```html
<fieldset>: 폼 내에서 <label> 과 같은 여러 컨트롤을 그룹화 하는데 사용된다.
<form>: 웹 서버에 정보를 제출하기 위한 대화형 컨트롤을 포함한 문서의 구획을 나타낸다.
<input>: 용자로부터 데이터를 받아들이기 위한 웹을 기초로 하는 폼의 상호적인 제어를 만들어 낸다.
    [type 값]
    text - 기본값. 한 줄의 텍스트 입력 칸을 만듦
    password - text 속성값과 같지만, 문자를 숨겨서 표시(별표 처럼)
    checkbox - 체크박스를 만듦
    radio - 라디오 버튼을 만듦
    button - 누름 버튼을 만듦. 
    submit - 전송 버튼을 만듦
    reset - 재설정 버튼을 만듦
    file - 파일 선택 창을 만듦.
    hidden - 사용자에게 보이지 않는 숨김 창을 만듦
    image - 이미지로 된 전송 버튼을 만듦 src 속성으로 이미지 url 지정
    
    checked - 페이지가 로드될 때 요소가 미리 선택되어 있음
    disabled - input 요소가 비활성화 상태
    readonly - 입력 영역이 읽기만 가능하게 지정
    autofocus - input 요소가 자동으로 포커스 됨
    multiple - 사용자가 하나 이상의 값을 입력할 수  있도록 지정
    placeholder - input 요소의 예상된 값을 설명하는 짧은 힌트
    required - 입력 영역이 양식 전송 전에 채워져야함을 지정.
<label>: 유저 인터페이스 내 아이템의 캡션을 나타낸다.
<legend>: 부모 요소 <fieldset>의 캡션을 의미한다.
<progress>: 진행률을 표시 줄로 표시된 작업의 진행정도를 보여준다.
    value: 현재 작업률의 값을 설정한다.
    max: 작업률의 최대 값을 설정한다.
<select>: 옵션들의 메뉴가 존재하는 컨트롤을 나타낸다. 목록의 내용은 option 태그로 만든다.
    autofocus : 페이지가 로드되었을 때 해당 목록으로 바로 포커스가 맞춰진다.
    disabled : 화면에는 보이지만 사용할 수 없도록 만든다.
    multiple : 여러 값을 선택할 수 있도록 한다.
    size : 목록을 펼치기 전에 보여질 목록의 개수를 정한다.
<textarea>: 멀티라인 일반 텍스트 편집 컨트롤을 나타낸다.
```



## 8. 대화형 요소

```html
<details>: 사용자가 추가 정보를 검색할 수 있는 공시 위젯으로 사용된다.
<summary>: 반드시 <details> 요소의 첫 번째 자식 요소이어야 하며, 클릭함으로써 <details> 요소를 보이도록 할 수도 있고 숨길 수도 있습니다.
```

```html
<details>
    <summary>오늘의 환율</summary>
    <ul>
        <li>달러($) : 1,135.90</li>
        <li>유로(€) : 1,284.08</li>
        <li>엔(￥) : 1,014.88</li>
    </ul>
</details>
```
